# VARS.js

Internal JavaScript utility UMD library.

# Usage

Add this repo to `package.json`:

```
{
  [..]
  "dependencies": {
    "vars": "git+https://github.com/VARIANTE/vars.web.git"
  }
  [..]
}
```
